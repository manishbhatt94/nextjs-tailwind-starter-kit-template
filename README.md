# NextJS TailwindCSS Typescript Template with PurgeCSS

## Getting Started

```
git clone https://github.com/muhajirdev/nextjs-tailwind-template
cd nextjs-tailwind-template

npm install
npm run dev
```

to build

```
npm run build

```

More info https://nextjs.org/learn/basics/getting-started
